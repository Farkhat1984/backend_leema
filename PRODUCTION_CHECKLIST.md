# ✅ Production Readiness Checklist

Используйте этот чеклист перед развертыванием Fashion AI Platform на production.

## 🔐 Безопасность

### Секреты и ключи
- [ ] `SECRET_KEY` изменен (не используйте дефолтный)
- [ ] `POSTGRES_PASSWORD` - сложный уникальный пароль
- [ ] `GOOGLE_CLIENT_SECRET` заполнен
- [ ] `GEMINI_API_KEY` заполнен и валиден
- [ ] `PAYPAL_CLIENT_SECRET` заполнен
- [ ] `SMTP_PASSWORD` заполнен (app password для Gmail)
- [ ] Все пароли сохранены в безопасном месте (password manager)

### Конфигурация
- [ ] `DEBUG=false` в `.env.production`
- [ ] `PAYPAL_MODE=live` (не sandbox!)
- [ ] CORS настроен только для реальных доменов
- [ ] `.env.production` НЕ в Git (в `.gitignore`)
- [ ] Файлы с паролями защищены: `chmod 600 .env.production`

### SSL/TLS
- [ ] SSL сертификаты получены для всех доменов
- [ ] HTTPS работает на всех доменах
- [ ] HTTP редиректит на HTTPS
- [ ] SSL сертификаты настроены на автообновление

### Firewall
- [ ] UFW/iptables настроен
- [ ] Открыты только порты 22, 80, 443
- [ ] SSH доступ ограничен (ключи вместо паролей)
- [ ] Fail2ban установлен и настроен (опционально)

## 🌐 Инфраструктура

### DNS
- [ ] A-запись для `leema.kz` → server IP
- [ ] A-запись для `www.leema.kz` → server IP
- [ ] A-запись для `api.leema.kz` → server IP
- [ ] DNS изменения применились (проверить через nslookup)
- [ ] TTL установлен разумно (300-3600 секунд)

### Сервер
- [ ] Минимум 2GB RAM (рекомендуется 4GB+)
- [ ] Минимум 2 CPU ядра
- [ ] Минимум 20GB диска (рекомендуется 50GB+)
- [ ] Ubuntu 22.04+ или аналогичная ОС
- [ ] Сервер обновлен: `apt update && apt upgrade`
- [ ] Часовой пояс настроен правильно

### Docker
- [ ] Docker установлен и работает
- [ ] Docker Compose установлен
- [ ] Docker daemon запускается автоматически
- [ ] Пользователь добавлен в группу docker

## 🔧 Приложение

### Конфигурация URLs
- [ ] `FRONTEND_URL=https://www.leema.kz`
- [ ] `GOOGLE_REDIRECT_URI=https://api.leema.kz/api/v1/auth/google/callback`
- [ ] CORS включает все production домены
- [ ] В Google Console добавлен redirect URI

- [ ] Volume для uploads настроен
- [ ] Бэкапы uploads настроены (опционально S3)

### API Services
